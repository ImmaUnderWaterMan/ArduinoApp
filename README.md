# Arduino Learning Hub 🚀

**Информационная обучающая система по созданию гаджетов на Arduino**

## 📖 О проекте

**Arduino Learning Hub** — это десктопное приложение (WPF), разработанное для удобного изучения основ работы с платформой Arduino. Система объединяет теоретические учебные материалы и интерактивный справочник электронных компонентов в одном интерфейсе.

Проект реализует архитектуру клиент-серверного приложения с локальной базой данных SQLite, разделением логики на слои (Data, Business, UI) и покрытием модульными тестами.

## ✨ Возможности системы

### 📚 Учебный курс
*   Структурированные уроки по темам: «Введение в Arduino», «Электроника и схемы», «Основы программирования C++».
*   Удобная навигация по списку тем с мгновенной загрузкой контента.
*   Поддержка форматирования текста для лучшей читаемости материалов.

### ⚡ Интерактивный справочник компонентов
*   База из **20+ популярных электронных компонентов**: датчики (DHT11, HC-SR04, PIR), актуаторы (LED, Servo, Relay), модули связи (Bluetooth HC-05, Wi-Fi ESP-01) и пассивные элементы.
*   **Карточка компонента** включает:
    *   Фотографию элемента.
    *   Технические характеристики (напряжение, количество контактов).
    *   Описание схемы подключения.
    *   Готовый пример кода (скетча) для Arduino IDE.
*   Кнопка **«Копировать код»** для быстрого переноса скетча в буфер обмена.

### 🔍 Поиск и навигация
*   Мгновенный поиск компонентов по названию (поддержка частичного совпадения).
*   Автоматическая подсветка найденного элемента в общем списке.
*   Дружелюбные уведомления об ошибках при отсутствии данных или некорректном запросе.

## 🛠 Технологии

*   **Язык:** C# (.NET 6 / .NET 8)
*   **Интерфейс:** WPF (Windows Presentation Foundation)
*   **База данных:** SQLite (пакет `Microsoft.Data.Sqlite`)
*   **Тестирование:** xUnit
*   **Среда разработки:** Visual Studio 2022 Community

## 📂 Структура решения

Решение состоит из трех основных проектов:

1.  **`ArduinoApp`** — Проект пользовательского интерфейса (WPF). Содержит разметку XAML и логику отображения.
2.  **`ArduinoCore`** — Библиотека классов (Class Library).
    *   `Data`: Работа с базой данных (`DatabaseService`).
    *   `Business`: Бизнес-логика (`ArduinoManager`).
    *   `Models`: Модели данных (`Topic`, `Component`).
3.  **`ArduinoCore.Tests`** — Проект модульных тестов (xUnit). Проверяет корректность поиска, расчетов и обработки данных.

## 🚀 Установка и запуск

### Требования
*   ОС: Windows 10/11

### Запуск из исходного кода
1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/ВАШ_НИКНЕЙМ/Arduino-Learning-Hub.git
    ```
2.  Откройте файл `ArduinoLearningHub.sln` в Visual Studio 2022.
3.  Нажмите **F5** или кнопку **Start** для запуска отладки.

### Запуск готовой версии (Release)
1.  Скачайте архив с релизной версией.
2.  Распакуйте содержимое в любую папку.
3.  Убедитесь, что рядом с файлом `ArduinoApp.exe` находятся:
    *   Файл базы данных `arduino_db.sqlite`.
    *   Папка `Images` с картинками компонентов.
4.  Запустите `ArduinoApp.exe`.
